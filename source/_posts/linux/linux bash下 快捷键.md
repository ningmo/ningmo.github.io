```
c + a    # 光标跳转到最左
c + e    # 光标跳转到最后
c + w    # 删除最后输入的单词
c + u    # 删除整行
c + k    # 删除光标到末尾
c + l    # 清屏

c + z    # 挂起当前进程至后台
         # jobs 查看挂起(相当与暂停)的进程
         # fg jobsId 切换至进程
         # bg jobsId 将挂起的进程 放至后台继续运行
```
